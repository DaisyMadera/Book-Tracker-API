# Book Tracker API

A simple Book Tracker application built with Ruby on Rails and Rails REST Framework.

This project allows users to view, add, edit, and delete books through a browser-based interface, while also supporting API-style routes. It also supports optional book cover image uploads.

## Features

- View all books in a styled table
- Add a new book
- Edit an existing book
- Delete a book
- Upload and display book cover images
- Custom options page for the books endpoint
- Rails REST Framework integration

## Ruby Version

- Ruby 3.4.9

## System Dependencies

- Ruby
- Rails
- SQLite3
- Bundler
